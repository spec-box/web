# SpecBox.WebUi

## Как запустить локально

1. `nvm use`
1. `npm ci`
1. `npm start`

Перед запуском нужно поправить [хост в конфиге](./vite.config.ts#L25) — укажите хост, на котором запущен [бэкенд](https://github.com/spec-box/api).
